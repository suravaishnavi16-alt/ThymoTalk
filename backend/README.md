# ThymoTalk Advanced Studio API

Emotion-aware voice assistant backend powered by FastAPI, Groq, and Deepgram.

## Setup on Hugging Face Spaces

1. Create a new Space.
2. Select **Docker** as the SDK.
3. In the Space **Settings**, add the following **Secrets**:
   - `DEEPGRAM_API_KEY`
   - `GROQ_API_KEY`
